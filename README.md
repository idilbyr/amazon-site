## How to Talk to Your Cat About Gun Safety - Website Project

This repository holds the website files for a project showcasing the satirical book "How to Talk to Your Cat About Gun Safety: And Abstinence, Drugs, Satanism, and Other Dangers That Threaten Their Nine Lives".

### Features

* **Responsive Layout:** The website adjusts seamlessly to various screen sizes, ensuring a great user experience on phones, tablets, and desktops.
* **Clean & Minimal Design:** The website prioritizes clarity and readability with a simple layout, typography, and color scheme.
* **Book Information:** The website displays the book's title, author, cover image, ratings, and a concise description.
* **Author's Amazon Page:** Clicking the author's name redirects you to their Amazon author page.

### Files

* **index.html:** This is the main HTML file that builds the website structure.
* **styles.css:** This file contains the CSS styles that define the website's visual appearance.

### Usage

1. Clone or download this repository.
2. Open the `index.html` file in your web browser to view the website.
